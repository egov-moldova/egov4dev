## Prequisites

To connect to MDelivery, the institution must meet the following conditions:

1. The institution is a public or private service provider that issues paper documents;
2. It has an information system registered in the [Registrul Resurselor și Sistemelor Informaționale](https://rsi.gov.md/procedure) that is available for integration with the MDelivery system;
3. It signs the contract for the delivery of the results of public services provided to beneficiaries;
4. It designates persons responsible for managing orders and users;
5. The designated persons have an electronic signature;
6. It defines the points of collection of documents by delivery service providers;

## Connection steps for the beneficiary

1. Submit a connection [request form](https://forms.office.com/pages/responsepage.aspx?id=Z4f8jWsRaEKDxfvIWTRtOHwLku9yOJZAs22J7gTUxBNUMFhISEJPMUNGOTIwV09OTkFTUUtLSk9LTS4u) available on the e-Government Agency website;
2. After the request is processed, the contract for using the MDelivery service is signed;
3. AGE configures the institution's access to the test environment:
4. The institution implements the API connection in accordance with the technical documentation and performs functionality tests;
5. Testing and validation of delivery flows and confirmation of technical compliance;
6. Activation in the production environment
